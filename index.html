Wocult interview approach
Set the ground → Explore the lived reality → Find the tension → Look for answers → End with meaning
Ask clearly. Celebrate what works. Challenge what doesn’t. Don’t dodge the uncomfortable question if that is where the real story sits.
Use this as a guide while framing questions. The interview does not need to be rigid, but it should move from context to real experience, challenge, solutions, and insight.
